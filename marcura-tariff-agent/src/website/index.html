Intelligent port tariff calculation. Any port. Any vessel.

Automated extraction, validation, and calculation of port tariffs powered by AI and deterministic logic.

View on GitHub

Port tariffs are complex. Calculation shouldn't be.

Manual tariff interpretation leads to errors and inconsistencies in calculation.

Rules vary dramatically by port, vessel type, and operational conditions.

No automated system handles the diversity of tariff document formats and structures.

HarbourMind reads the rules. You get the numbers.

Dynamic Rule Discovery

Intelligent agents discover all charges and conditions from any tariff PDF, no pre-configuration needed.

Confidence Scoring

Every extracted parameter carries a confidence level, flagging uncertainty and uncertain calculations.

Post-Hoc Refinement

Users can challenge any calculation; agents clarify the extraction and recalculate on demand.

Technology Stack

Built on

Gemini 2.5 Flash

Fast and accurate LLM for rule extraction and reasoning over tariff documents.

LangChain

Agent orchestration and tool-calling framework for structured extraction workflows.

FastAPI

Modern, async REST API framework for exposing calculations as web services.

Python

Deterministic calculator functions ensuring precise ZAR currency calculations (±0.01).

Pydantic

Schema validation and data integrity across all extraction, calculation, and API layers.

Docker

Multi-stage containerization for reproducible deployments across any infrastructure.

Run it yourself

1. Clone

git clone https://github.com/fauzanpolymath/harbourmind

2. Install

pip install -r requirements.txt

3. Run

docker-compose up
